解决: Error creating bean with name ‘dataSource’ defined in class path resource [applicationContext.xml]

1.错误日志如下：

2.原因及解决方法:

原因：

我再导入spring的jar包的时候，把aop和aspects的5.2.13的版本导入进来了，而其他的spring的jar包全是4.3.6版本的，应该是由于版本不一致导致的错误——创建类路径资源[applicationContext.xml]中定义的名称为“dataSource”的bean时出错。

解决方法：

我把这两个jar包换成了4.3.6的版本，程序就能正确运行了。
